Tutorial: IO Buster Commands

IO Buster Commands

IO Buster Commands

iob run

Runs a test script.

Options

  • --script: Required, Sets a path to a script file. This is relative to --workspace
  • --log: Sets a path to a log directory. Default: $PWD/log
  • --workspace: Sets a path to script workspace directory. Default: $PWD
  • --pid: Sets path to a process context file. Default: disabled.
  • --port: Sets a port to use. Default: 0.

iob interactive

Starts the interactive CLI.

iob scan

Scans available NVMe drives. Each scanned drive will be assinged to a port.

Options

  • --json: Prints all scanned ports in JSON.
  • --configure: Configures Grub to set cpu exclusions and huge pages.

iob ports

Prints total number of supported ports.

iob version

Prints an IO Buster version.

iob install

Installs dependencies and make this binary executable globally.

iob help

Shows usage.
