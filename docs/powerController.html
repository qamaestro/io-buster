(static) init(type, options)

Initializes Power Controller

Parameters:
Name Type Description
type string

Power Controller type. Available power controller types:

  • generic
options object

Power Controller options

Properties
Name Type Description
onScript string

Power on script path. This path is relative to the current script path.

This option is required for generic type.
offScript string

Power off script path. This path is relative to the current script path.

This option is required for generic type.
