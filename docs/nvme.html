(static) startWorkloadGenerator(optionsopt) → {nvme.WorkloadResult}

Generates IO workloads.

Parameters:
Name Type Attributes Description
options Object <optional>

Optional parameters

Properties
Name Type Attributes Default Description
nsid Number <optional>
 0

Namespace ID If nsid is 0, then the first available namespace will be used.
queueDepth Number <optional>
 0

Queue Depth If queueDepth is 0, then queueDepth will be the same as MQES.
type String <optional>
 basic

Type of workload generator Available type

  • basic
workload String <optional>
 sequential

Workload access pattern Available patterns:

  • sequential
  • random
range String | Array | Object <optional>

Access range

  • If a String value is given and if it ends with %, the workload generator will access total ranges based on percentage of the value provided. For example, if 70% is provided, then the workload generator will access 50% of total available ranges.
  • If a String value is given and if it ends with k,m,g,t, the workload generator will access total ranges based on bytes of the value provided. For example if 100g is provided, then the workload generator will access 100GB of total available ranges.
  • If an Object value is given, it must contain two propertes: startLba and size. Both startLba and size are based on LBA.
  • If an Array value is given, it contains a list LBAs.
readPercent Number <optional>
 100

Percentage of read access
alignment Number <optional>
 0

LBA alignment
xferSize Number <optional>
 0

Transfer size in bytes Must be multiples of 512. If 0, xferSize will be the maximum tranfer size for the given Namespace ID.
duration Number <optional>
 0

duration in seconds
loops Number <optional>
 1

Number of loops to repeat
dataPattern Buffer <optional>
 null

Cutomized data buffer for write commands. Must be multiple of 8 bytes (i.e., dataPattern: new Buffer.from([0x11, 0x22, 0x33, 0x44, 0x55, 0x66, 0x77, 0x88]))
ignoreError Array <optional>
 null

Ignored command errors during execution of WorkloadGenerator (i.e., ignoreError: [{sct:0x00, sc:0x80}, {sct:0x00, sc:0x80}])
